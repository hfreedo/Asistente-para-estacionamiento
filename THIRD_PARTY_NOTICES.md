# Dependencias de terceros

ParkAssist utiliza componentes de software desarrollados por terceros. La
Licencia MIT ubicada en `LICENSE` cubre el código propio del proyecto y no
reemplaza las licencias de estas dependencias.

## Adafruit NeoPixel

- Proyecto: Adafruit NeoPixel Library
- Uso: control de tiras WS2812/WS2812B desde Arduino.
- Repositorio: https://github.com/adafruit/Adafruit_NeoPixel
- Licencia: GNU Lesser General Public License v3.0 o posterior.

La biblioteca se instala mediante el administrador de bibliotecas de Arduino y
no se incluye como copia dentro de este repositorio.

## AndroidX DrawerLayout

- Artefacto: `androidx.drawerlayout:drawerlayout:1.2.0`
- Uso: panel lateral de navegación de la aplicación Android.
- Proyecto: https://github.com/androidx/androidx
- Licencia: Apache License 2.0.

El artefacto se descarga mediante Gradle desde los repositorios configurados en
el proyecto Android.

## Android, Kotlin y herramientas de compilación

Android SDK, Android Gradle Plugin, Gradle y Kotlin se utilizan como herramientas
o plataformas de desarrollo y conservan sus respectivas licencias y términos.
No se relicencian como parte de ParkAssist.
